# Model Files

The trained machine learning models used in this project are not stored directly in this GitHub repository because of GitHub file size limitations.

Instead, the model files are hosted on Hugging Face.

You can download the models from the following link:

https://huggingface.co/spaces/codedevrosh/multi_disease_prediction/tree/main/models

## Available Models

- heart_disease_model.pkl
- diabetes_model.pkl
- kidney_model.pkl
